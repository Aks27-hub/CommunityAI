This project enables community residents to report community infrastructural issues such as broken roads, damaged lights or potholes to the local authority so that timely action can be taken and the issues can be resolved in a transparent way. If the issue does not get solved then the matter escalates to a higher authority. The user also gains points for reporting issues and participating in community issues thereby encouraging community participation.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
